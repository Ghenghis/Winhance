## Category 1: Intelligent File Classification (10 Features)

### 1. Windows System File Detector
Automatically identifies Windows system files vs user files:
- Registry-linked files (installed apps)
- System32/SysWOW64 protected files
- Windows Update components
- Driver files
- **Safety Rating**: RED (never move), YELLOW (ask user), GREEN (safe)

### 2. Installation Footprint Mapper
Maps every installed application's file footprint:
- Executable location
- Registry entries
- AppData folders
- ProgramData components
- Temp file patterns
- **Uninstall Impact Analysis**

### 3. User-Created File Identifier
Distinguishes user content from system/app files:
- Documents you created vs downloaded
- Photos you took vs app-generated
- Projects vs dependencies
- Personal vs work content

### 4. Orphan File Detective
Finds files left behind by uninstalled apps:
- Dead registry references
- Abandoned AppData folders
- Leftover temp files
- Broken shortcuts
- **One-click cleanup with undo**

### 5. File Origin Tracker
Tracks where every file came from:
- Download source URL
- Creation application
- Copy/move history
- Network location origin
- Cloud sync status

### 6. Dependency Chain Analyzer
Maps file dependencies:
- DLLs required by executables
- Config files linked to apps
- Database connections
- Library dependencies

### 7. Active Usage Monitor
Tracks file access patterns:
- Last accessed timestamp
- Access frequency heatmap
- Application associations
- Peak usage times

### 8. Sensitivity Classifier
AI-powered content sensitivity detection:
- Personal information
- Financial documents
- Medical records
- Passwords/credentials
- **Auto-encryption suggestions**

### 9. Project Boundary Detector
Automatically identifies project folders:
- Git repositories
- Node.js projects (package.json)
- Python projects (requirements.txt)
- VS solutions
- **Never break project integrity**

### 10. Duplicate Intelligence
Smart duplicate detection beyond hash matching:
- Similar images (perceptual hash)
- Near-duplicate documents
- Version variations
- Backup copies vs intentional duplicates
